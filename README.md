(C#) 햄버거 키오스크

1줄 소개 : 원하는 메뉴를 고르고 주문하기 버튼을 누르면 자동으로 가격을 계산해주는 시스템

사용한 플랫폼 : c#, .Windows Forms, Visual Studio, GitHub

사용한 컨트롤 : Label, TextBox, Button, checkbox, groupbox

사용한 기술과 구현한 기능:
Visual Studio를 이용한 UI 디자인

원하는 메뉴를 직접 선택할 수 있는 시스템

부가 메뉴도 직접 고를 수 있음

모두 고르고 주문하기 버튼을 누르면 알아서 선택한 모든 메뉴의 가격을 합산하여 알려줌

새로운 메뉴를 고르고 싶을 때는 초기화 버튼을 누르면 모두 삭제됨
